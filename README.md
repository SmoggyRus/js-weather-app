# Weather App 🌤️

Простое веб-приложение для отображения текущей погоды в реальном времени. Получает данные через **OpenWeatherMap API** и предоставляет информацию о температуре, влажности и скорости ветра для любого города мира.

## Возможности

- 🔍 Поиск погоды по названию города
- 🌡️ Отображение текущей температуры (°C)
- 💧 Уровень влажности (%)
- 💨 Скорость ветра (km/h)
- 🖼️ Динамические иконки погодных условий
- ⌨️ Поддержка поиска через кнопку и клавишу Enter
- ⚠️ Обработка ошибок с полезными подсказками
- 📱 Адаптивный дизайн
- 🌐 Интерфейс на русском языке

## Технологии

- **HTML5** — семантическая разметка
- **CSS3** — кастомные свойства (CSS variables), Flexbox, адаптивная верстка
- **JavaScript (ES6+)** — `async/await`, Fetch API, DOM-манипуляции
- **OpenWeatherMap API** — источник погодных данных
- **Google Fonts** — шрифт Poppins

## API конфигурация

Приложение использует **OpenWeatherMap Current Weather API**:

- **Endpoint:** `https://api.openweathermap.org/data/2.5/weather`
- **Параметры:**
  - `units=metric` — температура в Цельсиях
  - `lang=ru` — описания на русском языке
  - `q={city}` — название города
  - `appid={API_KEY}` — ключ доступа

> ⚠️ **Важно:** API-ключ хранится в `script.js(Специально)

## Валидация

Приложение проверяет:
- Пустой ввод
- Допустимые символы в названии города (буквы, пробелы, дефисы)
- Ответ сервера (обработка 404 и других ошибок)
